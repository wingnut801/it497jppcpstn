File Server Migration

This project involved migrating a large dataset from legacy infrastructure to a secure, modern file system using PowerShell and Robocopy. Spiritually, this reflected my value of integrity and stewardship—handling sensitive systems with care and follow-through.

Spiritually, this project deepened my sense of vigilance and responsibility—watching over systems the way one might watch over others with care and diligence.
